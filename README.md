## Scan and generate QR-Codes easily
### Install Required libraries
``` bash
git clone https://github.com/coder12341/qr-tools.git && cd qr-tools
pip3 install -r requirements.txt
```

### Execute
``` bash
python3 QR-Tools.py
```
