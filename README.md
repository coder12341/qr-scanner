# qr-tools
QR-Code Scanner and Generator

## Scan and generate qr-codes easily
### Generate qr
```qr_generator.py -t 'Some Text' [-l logo.png] [--qrversion 1] [--boxsize 10] [--border 4] [--background white] [--foreground black] [--help]```

### Read qr from file
```qr_reader.py -i qr.png```

### Read qr (live)
```qr_reader_live.py```
